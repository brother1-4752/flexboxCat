Flexbox Cats는 CSS Flexbox를 이용해 고양이를 츄르가 있는 곳으로 이동시키는 게임입니다. justify-content 속성을 이용하여 고양이가 츄르가 있는 곳으로 이동시켜주세요. 이 속성은 다음의 값들을 인자로 받아 요소들을 가로선 상에서 정렬합니다.

  • flex-start: 요소들을 컨테이너의 왼쪽으로 정렬합니다.
  • flex-end: 요소들을 컨테이너의 오른쪽으로 정렬합니다.
  • center: 요소들을 컨테이너의 가운데로 정렬합니다.
  • space-between: 양 끝을 포함X, 요소들 사이에 동일한 간격을 둡니다.
  • space-around: 요소들 주위에 동일한 간격을 둡니다.
  • space-evenly: 양 끝을 포함O, 요소들 주위에 동일한 간격을 둡니다.

예를 들어, justify-content : center;는 고양이를 가운데로 이동시킵니다.
